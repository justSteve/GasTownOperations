# CrudEngine Executive Presentation

Self-contained reveal.js presentation for executive walkthrough of CrudEngine work.

## Quick Start

1. Open `index.html` in any modern browser
2. Use arrow keys or spacebar to navigate
3. Press `S` to open speaker notes view
4. Press `F` for fullscreen
5. Press `O` for overview mode

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` / `→` | Next slide |
| `←` | Previous slide |
| `S` | Speaker notes (opens new window) |
| `F` | Fullscreen |
| `O` | Overview mode |
| `Esc` | Exit overview/fullscreen |
| `B` | Pause (black screen) |

## Technical Notes

- Uses reveal.js 4.6.1 via CDN
- Uses Mermaid 10 for diagrams via CDN
- No local dependencies required
- Works offline after initial load (browser caches CDN resources)
